# CS50P: Solutions (Harvard University)
Личный архив решений курса CS50P 2024. Код перенесен из учебного репозитория me50 для формирования публичного портфолио.
Что внутри:
1. Решения еженедельных Problem Sets
2. Unit-тесты (pytest) для проверки логики
3. Примеры работы с ООП, Regex, API и т д
4. Финальный проект
Статус: Все задания проверены и приняты системой check50/submit50
